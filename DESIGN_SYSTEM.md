# Forge Operating Group - Design System

## Color Palette

### Public Landing Page (Light Theme)
| Role | Color | Hex |
|------|-------|-----|
| **Background** | Warm Cream | `#F7F4EF` |
| **Foreground** | Near Black | `#1A1A1A` |
| **Primary** | Teal | `#14B8A6` |
| **Muted Text** | Gray | `#666666` |
| **Border** | Warm Gray | `#E8E4DD` |

### Team Dashboard (Dark Theme)
| Role | Color | Hex |
|------|-------|-----|
| **Background** | Deep Charcoal | `#141414` |
| **Foreground** | Off-White | `#F0F0F0` |
| **Primary** | Vibrant Orange | `#FF6B35` |
| **Card** | Dark Gray | `#1E1E1E` |
| **Sidebar** | Darker | `#0F0F0F` |
| **Border** | Subtle Gray | `#333333` |
| **Muted Text** | Gray | `#888888` |

### Accent Colors
| Color | Use |
|-------|-----|
| Teal `#14B8A6` | Public site CTAs, buttons |
| Orange `#FF6B35` | Dashboard primary, highlights |
| Amber `#F59E0B` | Warnings, stars |
| Green | Success states |
| Red | Destructive/errors |

---

## Typography

| Type | Font | Usage |
|------|------|-------|
| **Sans** | Inter | Body text, UI |
| **Mono** | JetBrains Mono | Code, technical |
| **Display** | Inter | Headings |

### Styles
- Headings: `font-semibold tracking-tight`
- Border radius: `0.75rem` (12px)
