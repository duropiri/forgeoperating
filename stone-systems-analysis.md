# Stone Systems Website Analysis

## Color Palette
- Background: Cream/Beige (#F5F0E8 or similar warm off-white)
- Text: Dark charcoal (#1A1A1A)
- Accent: Teal/Green (#2DD4BF or similar)
- Secondary: Soft grays for cards

## Typography
- Clean sans-serif (likely Inter or similar)
- Large, bold headlines
- Generous line spacing
- Playful but professional tone

## Key Sections (In Order)
1. **Hero**: "5-Star Magic Review Funnel" with star icons
2. **Value Props**: 3 feature cards with icons
   - Automatic Follow-Up Reminders
   - Ask For Reviews In One Click
   - Stop worrying about bad reviews
3. **How It Works**: Step-by-step process
4. **Social Proof**: Star ratings, testimonials
5. **FAQ**: Accordion-style questions
6. **CTA**: Book a call / Get started
7. **Footer**: Simple, minimal

## Design Patterns
- Rounded corners on cards
- Subtle shadows
- Generous whitespace
- Icons with soft backgrounds
- Star ratings prominently displayed
- Playful, confident copy ("we're fired and extremely embarrassed")

## Copy Style
- Direct, no fluff
- Addresses pain points
- Uses "we'll" and "you" language
- Slightly playful/confident tone
- Short paragraphs
